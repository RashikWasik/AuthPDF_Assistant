##  RAG + SentenceTransformer (Embeddings) + ChromaDB (Vector Database) +  FastAPI + bcrypt + PostgreSQL + Session Cookies + Docker + Gradio + Deployment (Hugging Face Spaces)
___________________________

## If the Space is sleeping, please click “Restart this Space” and wait a few seconds for it to start.
___________________________

## Features:
1. Upload any PDF and chat naturally in English
2. Get accurate answers based only on the content of your uploaded PDF
3. Smart RAG pipeline using SentenceTransformer embeddings and ChromaDB
4. Secure user registration and login with bcrypt password hashing
5. Session cookie authentication for protected access
6. One username cannot be used twice
7. One-click logout directly from the chat screen
8. FastAPI backend with integrated Gradio chat UI
9. User data stored in PostgreSQL database
10. Fully containerized using Docker
11. Quantized Qwen 1.5B model optimized to run efficiently on CPU (no GPU required)
12. Clean error handling and clear status messages
13. Deployed on Hugging Face Spaces
